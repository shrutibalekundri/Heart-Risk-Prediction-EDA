# 📖 About the Project

This project explores a heart disease dataset using **Exploratory Data Analysis (EDA)** to identify patterns, trends, and key health indicators associated with heart attack risk. It also includes a basic machine learning model that predicts the likelihood of heart disease based on patient health attributes.

---

# 🎯 Project Objectives

* 📊 Explore and understand healthcare data through visualization.
* ❤️ Analyze factors influencing heart disease risk.
* 📈 Discover correlations between medical parameters.
* 🤖 Build a basic machine learning model for prediction.
* 📌 Extract meaningful insights to support data-driven healthcare decisions.

---
